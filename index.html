    How to Play

    PAGE 1 — OBJECTIVE

    PLAYERS MUST FIND AND SUBMIT A LIST OF 10 VALID WORDS WITH THE HIGHEST COMBINED SCORE

    PAGE 2 — TILE VALUES

    EACH TILE STARTS WITH A STANDARD BASE POINT VALUE

    • 1 point: A, E, I, O, U, L, N, R, S, T
    • 2 points: D, G
    • 3 points: B, C, M, P
    • 4 points: F, H, V, W, Y
    • 5 points: K
    • 8 points: J, X
    • 10 points: Q, Z

    PAGE 3 — TILE REUSE MULTIPLIER

    THIS BONUS APPLIES TO A TILES BASE POINT VALUE EACH TIME ITS BEEN REUSED

    • 1st REUSE: TILES BASE VALUE IS DOUBLED (×2).
    • 2nd REUSE: THAT VALUE IS DOUBLED AGAIN (×4).
    • MAXED OUT: AFTER 2nd REUSE THE TILES VALUE REMAINS MAXED OUT FOR ALL SUBSEQUENT USES

    PAGE 4 — WORD LENGTH MULTIPLIER

    WHEN THE LENGTH OF A WORD CONTAINS 5+ LETTERS

    • 5 LETTERS: ×3
    • 6 LETTERS: ×4
    • 7 LETTERS: ×5
    • 8 LETTERS: ×6
    • 9 LETTERS: ×7
    • 10+ LETTERS: ×10

    PAGE 5 — PHRASE PANEL / ANAGRAM BONUS

    ANAGRAM / PHRASE PAIR BONUS: .

    PAGE 6 — STRATEGY TIPS

    • PRIORITIZE LONG WORDS: THE LENGTH MULTIPLIER GIVES A HEFTY BOOST TO WORDS OF 5 LETTERS OR MORE
    • LOOK FOR REUSE OPPORTUNITIES: TAKE ADVANTAGE OF REUSING AND MAXING OUT TILES TO REAP THE FULL REWARDS
    • POOL OF POTENTIAL: YOUR WORD LIST PANEL COMES WITH AN IN GAME WORD SCORING CALCULATOR THAT WILL HELP YOU DETERMINE WHICH WORDS TO KEEP/REMOVE

    PAGE 7 — EXAMPLE CALCULATION

    HERE IS AN EXAMPLE USING QUIZ:

    • BASE VALUE: (Q=10) + (U=1) + (I=1) + (Z=10) = 22 Pts
    • REUSE BONUS: ASSUME THIS IS EACH TILES 1st USE SO NO REUSE MULTIPLIER
    • LENGTH MULTIPLIER: THE WORD HAS 4 LETTERS SO NO LENGTH MULTIPLIER
    • FINAL WORD SCORE: 22 Pts

    NOW, CONSIDER THE WORD QUIZZED & ASSUME BOTH Zs HAVE BEEN USED TWICE

    • BASE VALUE: Q=10, U=1, I=1, Z=10, Z=10, E=1, D=2
    • REUSE BONUS VALUE FOR Zs: EACH Z’s 10 Pts ARE MULTIPLIED BY 4
    • SUBTOTAL: 10 + 1 + 1 + (10×4) + (10×4) + 1 + 2 = 95
    • LENGTH MULTIPLIER: 7 LETTERS ×5 = 475 Pts TOTAL
